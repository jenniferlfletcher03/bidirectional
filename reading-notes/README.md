# Reading Notes

These are the notes from papers and books I've read as part of the bidirectional research project.

## Note format

Each note follows the same structure:

- **Title and source** — paper or book, author(s), year
- **What I understood** — the core argument or finding, in my own words
- **What I didn't** — what's still unclear
- **One question** — a specific question the reading raised
- **Threads it informs** — which parts of the project it connects to

## File naming

`author-year-short-title.md` (e.g., `long-sebo-2024-taking-ai-welfare-seriously.md`)
